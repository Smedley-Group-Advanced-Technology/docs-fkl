## Overview

This section covers how to use the Session Summaries functionality in SimTelLab to analyse kart battery health using data collected during charging.

## Prerequisites

Either of the following setups is required:

- **Developer licence:** MATLAB 2023b, 2024a, or 2024b with SimTelLab v12.92 or later
- **User licence:** Compiled SimTelLab v12.92 or later
