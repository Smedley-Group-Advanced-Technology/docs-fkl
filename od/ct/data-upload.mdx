## Overview

At the end of each day during an event, charging data must be exported from the Telemetry WebApp and uploaded to S3. This should be completed once per day.

<Warning>
  Do not make an export while monitoring the karts via the Battery Dashboard.
</Warning>

<Warning>
  The application must be running before attempting this. If you try to access the Telemetry WebApp without the application running, it will give you an error. See [Starting the Application](/od/ct/start) if needed.
</Warning>

---

## Steps

<Steps>
  <Step title="Open the Telemetry WebApp">
    Ensure the application is running, then open a browser and navigate to [http://localhost:4200](http://localhost:4200). You should see the Telemetry WebApp interface.
	
	![Home UI](/images/od/ct/upload-home.png)
  </Step>
  <Step title="Select the date">
    ![Select date](/images/od/ct/upload-step1-date.png)
  </Step>
  <Step title="Select the event type">
    In **Step 2**, select whether the event was a **Race** or a **Test**.

    ![Select event type](/images/od/ct/upload-step2-event-type.png)
  </Step>
  <Step title="Select the region">
    ![Select region](/images/od/ct/upload-step3-region.png)
  </Step>
  <Step title="Select the timezone">
    ![Select timezone](/images/od/ct/upload-step4-timezone.png)
  </Step>
  <Step title="Set the export threshold">
    The default is 5 minutes - this is the minimum run duration to be included in the export.

    ![Export threshold](/images/od/ct/upload-step5-threshold.png)
  </Step>
  <Step title="Press Export Charging Data">
    ![Export button](/images/od/ct/upload-export-button.png)
  </Step>
  <Step title="Review and confirm">
    A confirmation dialog will display a summary of the export configuration and the export path. Check that the configuration matches your expectation. If correct, press **Confirm Export**. If not, press **Cancel** and fix the configuration.
	
	![Export button](/images/od/ct/upload-confirm.png)
  </Step>
  <Step title="Wait for completion">
    The export and S3 sync will now run. A successful export will display the run data for each kart.

    ![Successful export](/images/od/ct/upload-success.png)

    <Warning>
      Do not close this page or select a new date while exports are ongoing. Doing so will cancel the remaining exports and the sync to S3.
    </Warning>
  </Step>
</Steps>
