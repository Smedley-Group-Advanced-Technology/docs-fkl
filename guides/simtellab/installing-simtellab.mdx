## Overview

This document covers how to install SimTelLab, its prerequisites, and check the installation has been successful. SimTelLab is required for all telemetry viewing, analysis, and lap time simulation.

<Note>
  This process requires you to have a MATLAB licence. If you do not have one, refer to the Installing SimTelLab Compiled.
</Note>

---

## Prerequisites

Before you begin, make sure you have:

- MATLAB 2024b
- Git
