## Overview

This is the Level 1 post-event maintenance checklist for **EK1 karts**. It must be completed after every event.

<Warning>
  All tasks marked Standard must be completed before a kart is returned to service. Non-standard tasks should be reported and scheduled for resolution.
</Warning>

---

## Demobilise

| Task | Duration (mins) | Standard |
|--|--|--|
| Remove transponders and return to charging case | 1 | ✓ |
| Remove ballast and return to storage box | 4 | ✓ |
| Remove all driver aids — seat foam and pedal extenders (where fitted, replace with standard pedals) | 2 | ✓ |
| Remove all circuit race numbers and return to allocated storage | 1 | ✓ |
| Fit factory magnetic identification numbers | 1 | ✓ |
| Remove all wheels, refit nuts and spacers to spindles hand tight | 8 | ✓ |
| Remove chain guard, check for damage and missing hardware | 1 | ✓ |
| Remove SevCon cover | 10 | ✓ |
