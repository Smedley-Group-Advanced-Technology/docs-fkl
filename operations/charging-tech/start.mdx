## Overview

Before each event day, you must start the application. This is done by running a batch script with no user input required.

---

## Prerequisites

- [Setup](/od/ct/setup) has been completed before the event weekend
- Access to the event laptop (NUC)

---

## Steps

<Steps>
  <Step title="Run the start script">
    On the Desktop, double-click `start_application.bat` to run it.

    ![Start script on desktop](/images/od/ct/start-desktop.png)
  </Step>
  <Step title="Confirm the application is running">
    Once the script completes, open a browser window and navigate to [http://localhost:4200](http://localhost:4200). You should see the Telemetry WebApp interface.

    ![Application running](/images/od/ct/start-running.png)
  </Step>
</Steps>

---
