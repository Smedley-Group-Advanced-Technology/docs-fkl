## Overview

This section covers the end-to-end process for running the charging monitoring application at an FKL event — from pre-event setup through to post-event backup and shutdown.

Telemetry data collected during the event is uploaded to S3 and ingested by the telemetry pipeline for analysis by the central hub.

## When to Run Each Step

| Phase | Task |
|--|--|
| Pre-event at factory or track | Setting up the application |
| Pre-event at track | Starting the application |
| In-event (end of each day) | Charging data upload |
| Post-event at factory or track | Stopping and backing up |
