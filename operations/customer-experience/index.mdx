## Overview

This section covers the key software tools used by CEL and event staff to manage FKL events end-to-end — from driver data and ticket management through to timing imports and driver sign-on.

<Note>
  Multiple people within the events team must know how to operate these tools to ensure appropriate cover and support can be provided to race weekend employees.
</Note>

## Access & Logins

Access to each platform is managed separately. Speak to your CEL or Hub Manager if you need an account created for any of the following:

- **Operational Dashboard** — use the **Contact Support** button on the OD
- **Alpha Timing** — use the **Contact Support** button on the OD
- **Sign On Website** — create an account and request approval
- **Vivenu** — use the **Contact Support** button on the OD
