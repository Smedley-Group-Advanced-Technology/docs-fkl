## CAN Bus Specification

### General Settings

| Detail | Value |
|--|--|
| Bitrate | 500 kbps |
| CAN ID Format | 11-bit hex |
| Transmit Period | 100 ms |

### Node ID to CAN ID Mapping

Node 0 is detailed below. Additional nodes 1–4 replace the first two digits as follows:

| Node ID | CAN ID Range |
|--|--|
| 0 | 0x100 – 0x10B |
| 1 | 0x180 – 0x18B |
| 2 | 0x200 – 0x20B |
| 3 | 0x280 – 0x28B |
| 4 | 0x300 – 0x30B |

<Note>
  Pin 5 = Battery Switch +VE, Pin 6 = Battery Switch -VE.
</Note>

### CAN Frame Definitions (Node 0)

<AccordionGroup>
  <Accordion title="0x100 — Total Voltage / Current / Remaining Capacity">
    | Bytes | Content | Unit | Notes |
    |--|--|--|--|
    | 0–1 | Total voltage | 10 mV | Unsigned (Big Endian) |
    | 2–3 | Total current | 10 mA | Signed (Big Endian), charge +, discharge − |
    | 4–5 | Remaining capacity | 10 mAh | Unsigned (Big Endian) |
    | 6–7 | CRC | — | Byte 0–5: CRC-16 calibration |
  </Accordion>

  <Accordion title="0x101 — Full Charge Capacity / Cycle Times / SOC">
    | Bytes | Content | Unit | Notes |
    |--|--|--|--|
    | 0–1 | Full capacity | 10 mAh | Unsigned (Big Endian) |
    | 2–3 | No. of cycles | 1 | Unsigned (Big Endian) |
    | 4–5 | Remaining SOC | % | Unsigned (Big Endian) |
    | 6–7 | CRC | — | Byte 0–5: CRC-16 calibration |
  </Accordion>

  <Accordion title="0x102 — Equalisation State / Protection Status">
    | Bytes | Content | Unit | Notes |
    |--|--|--|--|
    | 0–1 | Cells 1–16 balance status | Bit | 1 = open, 0 = closed |
    | 2–3 | Cells 17–32 balance status | Bit | 1 = open, 0 = closed |
    | 4–5 | Protection sign | Bit | 1 = protected, 0 = unprotected |
    | 6–7 | CRC | — | Byte 0–5: CRC-16 calibration |
  </Accordion>

  <Accordion title="0x103 — FET Control Status / Production Date / Software Version">
    | Bytes | Content | Unit | Notes |
    |--|--|--|--|
    | 0–1 | FET status | Bit | Bit 0 = charging FET, Bit 1 = discharging FET. 0 = off, 1 = on |
    | 2–3 | Production date | — | 2 bytes. Date = lowest 5 bits, month = next 4 bits, year = 2000 + remaining bits |
    | 4–5 | Software version | — | Signed (Big Endian) |
    | 6–7 | CRC | — | Byte 0–5: CRC-16 calibration |
  </Accordion>
