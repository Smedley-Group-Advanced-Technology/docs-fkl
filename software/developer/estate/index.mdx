The SG Advanced Tech software estate is documented here. This section covers each system's purpose, architecture, dependencies, and repositories.

## Architecture

![Software estate architecture diagram](/images/software/developer/software-estate.png)

The estate is split into two logical groups:

**Core estate** — The interconnected set of systems powering FKL events end-to-end, from driver booking through to timing data and telemetry.

**Fleet Management Platform** — Currently in development. Shares infrastructure dependencies with the core estate (via TalentIdPipeline-Infra-CDK for VPC) but has no shared data dependencies at this time. Integration with GKL Platform is anticipated.
